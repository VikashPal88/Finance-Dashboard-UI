# FinDash — Finance Dashboard UI

A clean, interactive, and visually premium finance dashboard. Track financial activity, explore transactions, manage multiple accounts, and gain spending insights — all from a beautiful, responsive interface.

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Start development server
npm run dev

# 3. Open in browser
# http://localhost:3000
```

> **Note:** If you see old data, clear your browser's localStorage.

## ✨ Key Features

- **Dashboard Overview:** View your total balance, income, expenses, and savings rate.
- **Transactions:** View, search, add, edit, and delete transactions.
- **Accounts System:** Manage multiple accounts (e.g., Main, Food, Travel) and set monthly budgets.
- **Insights & Analytics:** Visualize your spending trends with charts and statistics.
- **Dark/Light Mode:** Switch between dark and light themes easily.
- **Role-Based UI:** Toggle between Admin (can edit) and Viewer (read-only) modes.

## 🛠 Tech Stack

- **Framework:** Next.js & React
- **Language:** TypeScript
- **Styling:** Tailwind CSS & Shadcn UI
- **State Management:** Zustand (LocalStorage)
- **Charts:** Recharts
- **Animations:** Framer Motion
